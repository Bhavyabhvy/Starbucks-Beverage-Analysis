# Starbucks Beverage Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes Starbucks beverage nutrition data and provides insights into calories, sugar, caffeine, protein, and fat content across different beverage categories.

## Objectives

- Analyze nutritional characteristics of Starbucks beverages
- Compare beverage categories
- Identify high-caffeine beverages
- Explore relationships between sugar and calories
- Create interactive business intelligence dashboards

## Tools Used

- Power BI
- Power Query
- DAX
- Data Visualization

- Top 10 Highest Caffeine Beverages
- Sugar vs Calories Relationship

## Key Insights

- Coffee beverages have the highest average caffeine levels.
- Smoothies contain the highest protein levels.
- Sugar and calories show a positive correlation.
- Nutritional characteristics vary significantly across beverage categories.
